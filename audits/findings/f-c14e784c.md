## Title

Web Google login redirects to OAuth callback URL instead of initiating /auth/google

## Description

authApi.googleLogin assigns window.location to `${API_PUBLIC_URL}/auth/google/callback`, but the Nest AuthController exposes the Passport guard on GET `/auth/google` and uses `/auth/google/callback` only as the OAuth redirect target.

## Impact

Google sign-in from web breaks.
